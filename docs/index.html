What you're seeing

Map: Aftershocks from the 2019 Ridgecrest M7.1 sequence (USGS ComCat), colored by depth. Yellow lines show USGS Quaternary Fault traces.

InSAR analysis measures surface displacement to <1 cm precision using phase differences between radar images. The coseismic deformation field reached ~180 cm LOS displacement along the Airport Lake fault zone.

Processing Pipeline

1. Download LiCSAR pre-processed IFGs from COMET
2. Coherence mask threshold γ > 0.30
3. Phase unwrapping via SNAPHU MCF algorithm
4. Phase → displacement d = -λ/4π × φ
5. SBAS inversion linear + seasonal model
6. GNSS validation LOS projection, RMS misfit
